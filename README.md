# Registro De Usuarios Con Angular Y PHP

Este proyecto consiste en un sitio web para registrar, modificar, eliminar y consultar usuarios.

Página inicial.
Registro de usuarios.
Modificar usuarios.
Eliminar usuarios.
Reporte de usuarios.

## Tecnologías

- Angular para el frontend.
- PHP para el backend.
- MySQL para la base de datos.
- GitHub para trabajar en grupo.
